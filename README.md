# Web-development

Een **basispagina** bestaat uit volgende componenten:
* **HTML5 en XML** voor het structureren van gegevens
* **CSS/XSLT** voor het opmaken van gegevens
* **JavaScript (ECMAScript)** voor interactieve items en gedrag
